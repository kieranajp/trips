# AGENTS.md

Notes for AI agents. The code is small ‚Äî read it. This file is the stuff the code *won't* tell you: footguns and non-obvious wiring.

## The frontend is embedded in the binary

`main.go` does `//go:embed all:web`. Editing `web/*` has **zero effect** until you rebuild + restart. No dev server, no hot reload. Loop: edit `web/*` ‚Üí `go build` ‚Üí restart.

## "trips" (the app) vs "bilbao" (a trip)

`bilbao` is a *trip id*, not an old app name. `web/trips/bilbao.json`, its `index.json` entry, and `bilbao` DB/localStorage keys are **data** ‚Äî don't "rename bilbao ‚Üí trips". Only app-level identifiers (module, binary, chart, image) are `trips`.

## Two kinds of "data" ‚Äî never conflate

- **Trip definitions** ‚Äî committed static JSON in `web/trips/*.json`. The *seed*: catalogue, categories, neighbourhoods, starting flights/stays.
- **User state** ‚Äî pins, categories, flights, stays, per trip, in SQLite (`/state?trip=<id>`) + localStorage (`trip_state_<id>`). **Not in the repo.**

No separate "base" marker: home pin(s) render from whichever **stays** carry `lat`/`lng` ‚Äî a stay *is* its marker. Editing `bilbao.json` only reseeds a *fresh* trip; a user whose state has diverged loads their saved state, not the seed.

## SQLite is single-writer

Helm Deployment is `strategy: Recreate`, `replicaCount: 1`, RWO PVC. **Never set replicas > 1** ‚Äî two pods on one file corrupts state. Sync is naive last-write-wins (debounced PUT, pull on boot + window focus).

## The frontend has no build step

Preact + htm + `@preact/signals` via `<script type="importmap">` from esm.sh:
- **No JSX** ‚Äî `html\`...\`` tagged templates (htm).
- `?external=preact` on the signals/htm CDN URLs is **load-bearing** ‚Äî it forces one shared Preact instance; drop it and hooks break.
- **Leaflet is imperative, outside the Preact tree**, driven by signal `effect()`s in `web/features/map/leaflet.js`. `#map` is a `ref`'d escape hatch. Don't render markers through components.

## Tests run without npm install

`go test ./...` for the server; `npm test` for the frontend (Node 22+, built-in `node --test`, zero dependencies). The browser resolves `@preact/signals` via the import map in `index.html`; Node can't, so `tests/js/loader.mjs` maps that one specifier to a tiny local stub (`tests/js/stubs/preact-signals.js`). Pure logic modules are covered, and the Leaflet controller (`createTripMap`) is tested against a recording fake of the Leaflet API (`tests/js/leaflet.test.js`) ‚Äî Component/JSX code isn't. `package.json` exists only for `"type": "module"` + the test script; don't add runtime deps to it.

## The PWA service worker is network-first ‚Äî keep it that way

`web/sw.js` exists for installability + offline fallback, **not** speed: every push to `main` deploys, so a cache-first shell would serve stale UI forever. It never intercepts non-GET, API paths (`/state`, `/files`, `/expand`, `/whoami`, `/login`, `/health` ‚Äî auth redirects must reach the network), or map tiles (unbounded storage). Icons in `web/icons/` are generated by `go run ./cmd/genicons` (one-off tool, like cmd/import) ‚Äî regenerate rather than hand-edit, and keep `manifest.json` paths in sync (`TestPWAAssetsEmbedded` checks this).

## Trip switching is a full page load

`?trip=<id>` navigates via a real reload (on purpose ‚Äî sidesteps Leaflet teardown). No in-app trip swap. `?trip` must match `^[a-z0-9-]+$` or the server 400s.

## Secrets are gitignored ‚Äî the repo is PUBLIC

`.env` (Google Places key), `Takeout/`, `*.csv`, `bilbao-import.json` are gitignored. **Never commit them.** Before `git add -A`: `git ls-files --cached | grep -iE '\.env|Takeout|\.csv|import\.json'` must be empty.

## cmd/import & cmd/seedurls are one-off local tools

**Not** part of the running app. They read the gitignored CSV, call Google Places (`GOOGLE_PLACES_API_KEY`), and emit JSON you load via the Import button. Don't wire them into the server.

## Deploy is automatic on push to `main`

CI builds/pushes an **amd64** image `sha-<short>` + `latest` to GHCR; Deploy joins Tailscale + `helm upgrade --install trips`, pinning the exact `sha-` tag via `--set app.image.tag` (`:latest` + `Recreate` alone won't re-pull). Helm pinned to **v3** (v4 `--wait` false-times-out on a ClusterIP Service). Any push to `main` (docs included) deploys.

## Auth is at the ingress by method+path, not in Go

No auth code in `main.go` ‚Äî handlers trust that traffic already cleared Authentik. Access lives in `charts/trips/templates/ingressroute.yaml` (Traefik `IngressRoute`), decided by rule **priority**:
- **All GET/HEAD/OPTIONS are public** (priority 80). A new read endpoint returning private data is **public by default** ‚Äî gate it with a higher-priority `PathPrefix(...)` rule + `auth-private` middleware (see `/files`, priority 85).
- **All other verbs default-closed** (catch-all has `auth-private`) ‚Äî new writes gated automatically, new *reads* not.

Frontend `editable`/`canEdit` is **cosmetic**; the ingress is the trust boundary. Never rely on the SPA hiding a control to protect data.

## readOnlyRootFilesystem in prod

Container can only write to `/data` (the PVC). Keep SQLite + its temp/WAL under `DB_PATH` (in `/data`); writes elsewhere pass locally, fail in prod.
